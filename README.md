# tweet-scraper-using-links
Description: Simple python script to extract tweet content, impressions, likes, comments, replies and post date using a csv file of tweet links.

Welcome to yoda's Twitter Metrics Extractor Tool

Steps to setup:

1. Install Python
2. Install the needed libraries (your IDE should tell you)
3. Open "links.csv" in excel and paste all your twitter post links
4. Run using command: python3 main.py --input links.csv --output output.csv --failed failed.csv --email 'youremail@gmail.com' --username '@yourusername' --password "yourpassword"
